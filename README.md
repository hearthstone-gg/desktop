#Heathstone.gg Desktop App

##Install

Build Dependancies

`npm install`


App Dependancies

`cd app`

`npm install`



##Build

`grunt build`

##app events fired
``friendly-hero`` when the friendly hero is detected

``opposing-hero`` when the opposing hero is detected

``game-start`` when a game starts

``game-over`` when a game ends
